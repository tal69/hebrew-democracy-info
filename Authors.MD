# Preferred Authors For Nightly Scans

Use this file to list researchers whose new publications should receive priority in the daily paper scan.

The nightly Codex automation should read this file before searching for new papers. If it finds recent, relevant, non-duplicate publications by authors listed here, it should prefer those papers over equally suitable papers by unlisted authors. Listing an author here does not mean every paper by that author should be added; the paper still needs to fit the site's democracy, rights, institutions, rule-of-law, or liberal-democracy scope.

This repository is public. Add only information that can safely be public, especially for email addresses.

## Editing Notes

- Use `high`, `normal`, or `low` in the priority column.
- Hebrew and English names are both useful. Add both when possible.
- ORCID, official profile URLs, and stable affiliation text help the automation avoid mistaken identity matches.
- Leave unknown fields blank rather than guessing.
